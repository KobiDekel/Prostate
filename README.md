מחקרים קליניים מהשנים האחרונות מראים בעקביות: חומרים מסוימים במזון — ליקופן, סולפורפן, EGCG, כורכומין, אליצין — חודרים לרקמת הגידול, מעכבים את חלוקת התאים הסרטניים, וחותכים את אספקת הדם שממנה הגידול ניזון.
זו לא רפואה אלטרנטיבית. זו ביוכימיה מוכחת.
Gleason 3+3 ו-3+4 הם שלב שבו לזמן יש ערך מכריע. כל יום שהגידול גדל לאט יותר — הוא יום שנקנה. כל ארוחה נכונה היא מינון. כל שבוע של תזונה עקבית הוא טיפול מצטבר.
ערמונית מייצרת עבורך את "המינון" הזה מדי יום — תפריט שמשלב את כל החומרים הפעילים, במינון הנכון, בשילובים שמגבירים ספיגה.
הגידול לא יודע שאתה בחרת לאכול נכון -  אבל הגוף שלך כן.
